# Real Estate Website - React + Vite

A modern **React frontend** for a real estate platform built with **Vite**. Users can browse, search, and explore properties through a **simple, responsive, and user-friendly interface**.

---

## Features

- Browse and explore residential and commercial properties
- Responsive UI for desktops and mobile devices
- Property search and filter functionality
- Interactive property listings with details
- Built with React and Vite for fast development and hot module replacement (HMR)
- ESLint configured for code quality
